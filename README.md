# Multi-Agent Translation Evaluator

**Система межагентной оценки качества машинного перевода (англ → рус)**

Инструмент для объективной оценки качества машинного перевода с помощью нескольких специализированных LLM-агентов.

---

## 🎯 Зачем это нужно?

Традиционные метрики (BLEU, COMET и т.д.) плохо коррелируют с человеческим восприятием.  
Этот проект использует **ансамбль специализированных агентов**, каждый из которых оценивает перевод со своей экспертной точки зрения.

## ✨ Основные возможности

- **6+ специализированных агентов**:
  - Лингвист
  - Семантист
  - Терминолог
  - Прагматик
  - Культурный адаптер
  - Тематические эксперты (медицина, юриспруденция, стилист)

- **Модератор** — синтезирует мнение всех агентов и выносит финальную оценку
- Полная прозрачность: JSON-ответы от каждого агента
- Сохранение всех промежуточных результатов
- Подробный статистический анализ результатов
- Генерация отчётов и визуализаций

3. Как запустить пайплайн
Шаг 1. Установите зависимости:
Bashpip install -r requirements.txt
Шаг 2. Создайте файл .env в корне проекта:
YANDEX_API_KEY=your_api_key_here
YANDEX_FOLDER_ID=your_folder_id_here
Шаг 3. Положите файл translations.csv в папку data/
Шаг 4. Запустите оценку:
Bashpython main.py

5. Режимы работы
При запуске main.py вам будет предложено выбрать режим:

Полный анализ — оценка всех переводов (рекомендуется)
Тестовый запуск — 3 случайных сегмента (для проверки)
Выборочный анализ — указать количество сегментов


5. Что будет в результате
После завершения в папке results_full/ появятся:

full_evaluations_*.json — полные ответы всех агентов
evaluations_summary_*.csv — сводная таблица
visualizations/ — графики
analysis_report_*.txt — текстовый отчёт
